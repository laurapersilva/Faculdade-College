# JOGO SCRATCH
Laboratorio de Iniciação à Programação

Break Out!! 

Relatório do Projeto:

Introdução:
O jogo "Break Out!" foi desenvolvido como parte do projeto final em Scratch do professor João Paulo Aramuni. Segue meu relatório sobre a totalidade do jogo:

1. Descrição do Jogo:
"Break Out!" é um jogo em que você é um preso que encontra uma bola de concreto e a usa para sair da cadeia quebrando os tijolos. A principal mecânica do jogo envolve uma bola que rebate na plataforma controlada pelo jogador e, quando encontra com tijolos, os quebra.

2. Elementos Utilizados:

° Movimentos: Usei os blocos de movimento para controlar a movimentação da plataforma horizontalmente na base do cenário, também precisei deles para prever o movimento de rebatição da bola contra a própria plataforma e os tijolos.

° Aparências: Criei diferentes atores para representar o título, a plataforma (e sua fantasia que é o botão de start do jogo), o tijolo (que possui clones), a bola e as vidas.

° Sons: Eu mesma fiz meus sons de impacto da bola com a plataforma e da bola com os tijolos, além de também ter feito a música de comemoração ao jogador ganhar o jogo.

° Eventos: Utilizei eventos para inicializar o jogo e seus comandos "Start" e para avisar o que fazer quando um tijolo for destruído "sumiu".

° Controle: Os controles do jogo são simples, o jogador apenas precisa arrastar o cursor para um lado e para o outro para fazer a plataforma arrastar horizontalmente.

° Variáveis: Utilizei variáveis para 

      -Controlar o número de tijolos e, a partir disso, controlar a mudança de fases.
      
      -Controlar as vidas.
      
      -Mostrar os pontos feitos pelo jogador.
      

° Operadores: Utilizei operadores para diversas coisas como calculo de vida, calculo de tijolos, direção para onde a bola vai, controle da plataforma, etc.

3. Motivação:
Minha principal motivação para criar "Break Out!" foi a raiva, pois achei um jogo bem desafiador e estressante de criar. Quando comecei, me recusei a desistir. 

4. Funcionamento do Jogo:
O jogo começa ao jogador apertar "Start" e vai para o primeirio cenário, uma cela. Após controlar a bola para que ela acerte todos os tijolos, o cenário muda para o corredor de uma prisão e então, um muro. Se os tijolos do nível muro forem quebrados, passará para o cenário final, uma ravina verde com um lago, representando o seu escapamento. O jogador terá 3 vidas para quebrar a totalidade de 72 tijolos (24 por nível), caso perca todas, um policial o descobrirá e ele perderá o jogo.
